== Description ==

An availability calendar for vacation rentals driven via the iCal standard by any calendar that exports iCal format (HomeAway, VRBO, Flipkey, AirBnB, Google, Apple iCloud, and more). 
The availability calendar shows the next 36 months of availability. Shortcodes enable you to display your calendar on any page or post in three different sizes. Arrive and depart dates
from your master calendar are displayed as available and unavailable nights. The depart day is not counted as an unavailable night. Compatible with VRBO and HomeAway.

== Installation ==

1. Upload the contents of the zip to the '/wp-content/plugins/' directory.
2. Navigate to the 'Add New' in the plugins dashboard
3. Search for 'wp_vrcalendar_free_repo'
4. Click 'Install Now'
5. Activate the plugin on the Plugin dashboard
